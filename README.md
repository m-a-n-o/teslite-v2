# teslite-v2
Refactor version of TESlite


## System features
- TEI
  - Legal Basis
  - Mabuhay Lane, IRD, CTD
- NON-TEI
  - Category
  - Legal Basis (optional)
- PDF Form generation
  - Indorsement
  - DOF-RO Form 91
  - Order of Payment
  - Cancellation/Withdrawal Form
- Email Notification
  - Registration
    - Approve
    - Disapprove
- Registration
  - Consignee
  - Broker
  - Representative


## System technology
- SQL Server
- PHP
- Javascript
- HTML
- CSS
